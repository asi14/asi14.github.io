Fields, Variables, and Parameters

public class StudentTest
{
 public static void main(String args[])
 {
  Student stud1 = new Student("Tom", 16);
  Student stud2 = new Student("Sue", 15);
  stud1.showData();
  stud2.showData();
 }
}
Public class Student
{
  //these are called FIELDS, and are attributes of objects instantiated from the Student class.
  //you can also think of them as "placeholders" to be plugged in when the object is created.
  private String name;
  private int age;

  //String n and int a these are examples of PAREMETERS, and are placeholders for the programmer to plug in
  public Student (String n, int a)
  {
    name = n + " Cat:";
    age = a + 100;
  }

  public void showData()
  {

    //this is an example of a LOCAL VARIABLE: it exists within the method showData() only
    //once the program is done with showData(), then oldAge will be deleted
    int oldAge = 1000;

    for (int k = 1; k < 5; k++)
    {
      oldAge = oldAge + age;
    }
     System.out.print(name + " " + age + "!  ");
  }
}

Primative Data Types

Primative Data Types: they represent basic bits of information, and are most commonly seen in local variables. There are eight types, half required by AP CS, half not.
